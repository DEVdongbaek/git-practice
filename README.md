# git-practice

이 프로젝트는 git을 연습하기 위한 저장소입니다.

See [Demo](https://www.google.com/)

## Documentation


### Installation

```shell
$ git clone {repo url}
$ cd {reponame}
$ pip install -r requirements.txt
```

### How To Start
```shell
$ python main.py
```


### Supported Python versions
`>=3.6`
